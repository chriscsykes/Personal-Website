InSight App
Java

For CS 65 Smartphone Programming, I worked in a group of three to create an Android application called InSight, which was aimed at improving the quality of life of the visually imapaired. Our app recieved the Honorable Mention Award as one of the top three in the class. Using CameraX and Google Firebase, this app helps the visually impaired by providing features like:

-User friendly UI to log users in via speech

-Audio feedback on button presses using Google’s text to speech engine

-Firebase ML Kit’s text recogintion API to recognize text and provide audio feedback

-Firebase ML Kit’s image recognition API to recognize objects in the users surrounding, and provide audio feedback

-Add-on features like ‘Read Tutorials’, and ‘Provide Feedback’ that improve the users experience

*For more info and to download the APK, please visit our website: https://chriscsykes.github.io/InSight/

Tiny Search Engine
C and Unix

In this project, I created a Tiny Search Engine that takes a search query from the command-line and returns a list of documents that those words appear in. This project consisted of three main parts:

A "Crawler" which crawls a website and retrieves webpages starting with a specified URL. It parses the initial webpage, extracts any embedded URLs and retrieves those pages, and crawls the pages found at those URLs.

A "Indexer" which reads the document files produced by the Crawler, builds an index, and writes that index to a file.

A "Querier" which reads the index file produced by the Indexer, and page files produced by the Crawler, and answers search queries submitted via stdin. The output is a list ranked form highest to lowest count, showing the score, doc_ID, and URL.

Maze Solver
C and Unix

In this project, a few of my peers and I created a client-server application that simulates players --avatar threads-- searching for each other in a maze generated and managed by a server. The application was able to solve mazes of varying difficulty without any prior knowledge. To do this, our team developed a algorithm that implemented a breadth-first search to find dead ends, and a priority queue to store potential moves. In addition, the application included a visual user interface and a log file to track the avatars’ success/failures.

Dartmouth Pathfinder
Python

In this program, I created a graph that models Dartmouth's campus, finds the shortest path (least amount of vertices) from one point on campus to another, and displays that path. The user selects a starting vertex, and every vertex the mouse hovers over subsequently will display the shortest path from that vertex back to the start vertex. In order to find the shortest path between vertices, my program makes use of the Breadth-First search sorting algorithm with backchaining.

Huffman File Compression
Java

In this project, I created a program that implements a popular form of lossless file compression, Huffman encoding, to compress and decompress files. Instead of using 7 bits to encode each chararcter as ASCII does, Huffman encoding uses variable length encoding in which the length of each characters representation is directly related to its frequency in the text. In order to determine each character's code, a binary tree is created with each leaf representing a character, and each edge on the tree representing either a 0 or 1. The path from the root to that character gives that character's code word. My program was able to achieve roughly 50% file compression.
